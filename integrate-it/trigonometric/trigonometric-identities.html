Trigonometric Identities

Use trigonometric identities to rewrite the integrand. Often, after simplifying, we can integrate by substitution.

Basic Integration Formulas

Exponential

∫ e u du = e u + C

Powers

  • ∫ u n du = u n+1 n+1 + C , for n≠ -1
  • ∫ u -1 du = ∫ 1 u du = ln |u| + C

Trigonometric

  • ∫ sin u du = - cos u + C
  • ∫ cos u du = sin u + C
  • ∫ sec 2 u du = tan u + C
  • ∫ secu tanu du = sec u + C

Inverse Trigonometric

  • ∫ 1 1 + u 2 du = arctan u + C
  • ∫ 1 1 - u 2 du = arcsin u + C
