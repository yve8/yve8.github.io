Integrate It!

A guide to choosing a technique and evaluating an integral

Numerical Integration

Not every integral can be evaluated exactly. For many definite integrals, the value can only be approximated, by computer or by hand.

Calculator

If you are allowed to use a calculator, there are many resources to approximate definite integrals. Two free online options are:

  • Wolfram Alpha Integral Calculator
  • Desmos

No Calculator

If a calculator is not allowed, you may try approximating the integral using geometry. Some methods includ:

  • Left, Right, and Midpoint Riemann Sums (Mathematics LibreTexts)
  • Trapezoid Rule and Simpson’s Rule (Paul’s Online Notes)
Basic Integration Formulas

Exponential

∫ e u du = e u + C

Powers

  • ∫ u n du = u n+1 n+1 + C , for n≠ -1
  • ∫ u -1 du = ∫ 1 u du = ln |u| + C

Trigonometric

  • ∫ sin u du = - cos u + C
  • ∫ cos u du = sin u + C
  • ∫ sec 2 u du = tan u + C
  • ∫ secu tanu du = sec u + C

Inverse Trigonometric

  • ∫ 1 1 + u 2 du = arctan u + C
  • ∫ 1 1 - u 2 du = arcsin u + C
